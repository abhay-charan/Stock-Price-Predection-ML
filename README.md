📈 Stock-Price-Predection-Using-Machine-Learning

📌 Project Overview :
This project focuses on predicting stock prices using machine learning techniques and technical indicators.

🛠️Tools Used :
Python,
Pandas & Numpy,
Scikit-learn, 
Matplotlib and
Technical Indicators.

⚙️Process :
Data Collection and proessing, Feature enginerring using technical indicators, Model building & training and Prediction & evaluation.

##Key Insights :
Identified patterns in stock price movements, Applied machine learning models for predection, Evaluated model performance.

##Files Included :
Dataset (CSV file), 
Jupyter Notebook (Model + Analysis), 
Visualization outputs and
Final report.

##Conclusion :
This project demonstrates the application of machine learning techniques in finanial data analysis and predection.
